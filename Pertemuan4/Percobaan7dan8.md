#  Percobaan 7 dan 8

## 7. Buat subdirektory work dan play.
Gunakan perintah mkdir (make directory) untuk membuat folder baru di dalam direktori kita saat ini (Home):
```
mkdir work play
```
lalu ketik:
```
ls -F
```

## 8. Menghapus Subdirektori (rmdir)
Untuk menghapus folder yang masih kosong, kita menggunakan perintah rmdir (remove directory).
```
rmdir work
```
Lalu Verifikasi:
```
ls -F
```
