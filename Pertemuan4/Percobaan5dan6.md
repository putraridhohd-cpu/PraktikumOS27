## 5.	Ubahlah direktory home ke user lain secara langsung menggunakan cd ~username.
Pertama tama kita ketik kode ini untuk membuat user baru:
```
sudo adduser usernam
```
disini saya akan menggunakan nama depan saya yaitu "valent"
lalu cek siapa aja user yang ada di server dengan:
```
ls /home
```

## 6.	Ubah kembali ke direktory home saya

```
cd ~username
```
