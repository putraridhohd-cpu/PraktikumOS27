# mengganti semua kata server menjadi node pada file
## 1.sebelum mengganti
```
server started successfully
backup server running
server connection established
monitoring server active
```
## 2 Perintah yang digunakan:
```
sed -i 's/server/node/g' latihan
```

## 3.setelah mengganti
```
node started successfully
backup node running
node connection established
monitoring node active
```
